grinnell

An R package for dispersal simulations based on ecological niches

Fernando Machado-Stredel, Marlon E. Cobos, A. Townsend Peterson
Sep 22, 2023 Software

What is grinnell?

grinnell is an R package to simulate dispersal and accessibility based on niche estimations. One of the main functionalities is the simulation of species accessible areas (M), which can be used as calibration areas in Ecological Niche Models (ENM) and Species Distribution Models (SDM). These tools are based on the methods proposed by Machado-Stredel et al. (2021).


Check the grinnell package

If you want to try this tool you can install it on R from CRAN using remotes::install_github("fmachados/grinnell"). An example of its use can be found here.
